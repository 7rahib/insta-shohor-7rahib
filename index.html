Question 1
How does JavaScript work ?

JavaScript is a single-threaded interpreted high-level programming language that is used in web development. It's used to make a website more interactive. It runs on the client-side which means it is executed in the client's web browser. It can be easily be added using a script tag or we can add another file and connect it to our HTML page. It runs one thing at a time. The interpreter in the browser reads the whole JavaScript code, interprets each line at a time, and then runs it.

Question 2
What does JavaScript event loop do ?

JavaScript is a single-threaded interpreted language which means it has event loops that add functions from the execution queue to the execution stack. That means javascript does not execute functions parallelly. So, basically event loop is like gears used in a car. It checks if the execution stack is empty if empty then it pushed one function to the queue. And it waits until the queue is empty again to push another function.

Question 3
What are the differences between Local Storage and Session Storage ?

Local Storage is a property that helps a website to store or retrieve data in the form of key and value in the web browser and the data is never deleted. It has no expiration time. On the other hand,Session Storage is a property that does all the work as the Local Storage but it has an expiration date, it removes all the data as soon as the browser tab is closed. So, the main difference between Local Storage and Session Storage is that Local Storage does not expire but the Session Storage data is cleared when the session ends.
